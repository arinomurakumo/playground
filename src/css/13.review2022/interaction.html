Scroll Snap

スナップ点が存在する場合にスクロールコンテナーにどれだけ厳密にスナップ点を強制するかを設定

scroll-snap-children1
scroll-snap-children2

overscroll-behavior

スクロール領域の境界に達したときにブラウザーが何をするかを設定

overscroll-behavior: none;
止めた感じ
overscroll-behavior: contain;
跳ね返り

touch-action

タッチ画面のユーザーが要素のある領域をどのように操作できるか（例えば、ブラウザー内に組み込まれたパンまたはズーム機能）を設定

scroll-behavior

ナビゲーションや CSSOM のスクローリング API によってスクロールするボックスにスクロールが発生した際の、そのスクロールの振る舞いを設定

scroll-behavior: auto;
瞬時にスクロール
scroll-behavior: smooth;
滑らか

scrollbar-gutter

スクロールバー用のスペースを確保

scrtollbar-gutter: stable
クラシック スクロールバーを使用している場合、ボックスがオーバーフローしていなくても、オーバーフローが auto、scroll、または hidden の場合、ガターが表示されます。 オーバーレイ スクロールバーを使用する場合、ガターは表示されません。
