::marker

::marker は CSS の擬似要素で、リスト項目の箇条書き記号ボックス (ふつうは黒丸や番号) を選択します。これは [display: list-item](/ja/docs/Web/CSS/display) が設定された要素や擬似要素、例えば li 要素や summary 要素で利用できます。

:where()
